## 🧑🏻‍💻 Tecnologias

Projeto desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma

## 💻 Projeto

Adicionar e remove produtos, sendo que os itens podem ser marcados como comprados. Estrutura responsiva para diferentes tamanhos de telas.
